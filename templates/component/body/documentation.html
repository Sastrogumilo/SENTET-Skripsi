SENTIMEN ANALISIS TENTANG TWITTER (SENTET)
Program ini dibuat untuk melengkapi Skripsi yang berjudul SENTIMEN ANALISIS MULTI-LABEL PADA UJARAN KEBENCIAN DAN BAHASA KASAR DI TWITTER INDONESIA MENGGUNAKAN PENDEKATAN DEEP LEARNING
SENTET adalah sebuah program yang berguna untuk menganalisa sebuah kalimat di Twitter. Kalimat akan di kelompokan menjadi 4 kriteria, yaitu: Kategori Ujaran, Target Ujaran, Jenis Ujaran, dan Level Ujaran.

Kategori Ujaran : Abusive (kata kasar), HS (ujaran kebencian), dan Normal Target Ujaran: Individu dan Kelompok Jenis Ujaran: Menyingung agama, menyingung ras, menyingung fisik, menyingung gender dan menyingung jenis yang lain (other) Level Ujaran: Weak (lemah), Moderate (menengah), dan Strong (kuat)
Model klasifikasi yang digunakan untuk program ini menggunakan LSTM dengan Word2Vec. Model ini dipilih karena mendapatkan akurasi terbaik berdasarkan hasil penelitian Skripsi ini. Model dilatih pada Google Colabboratory menggunakan library Python berupa Tensorflow versi 2.0.0
Hasil dari pelatihan dapat dilihat pada gambar berikut

Pada graf diatas model mendapatkan hasil terbesar yaitu, 91,56%. Tetapi model tersebut mengunakan metode evaluasi Binary Akurasi
Dapat dilihat pada confussion matrix-nya, terdapat banyak True Negative yang berakibatkan model mendapatkan hasil binari akurasi yang tinggi
Oleh sebab itu peneliti menggunakan metrik akurasi yang bernama Hamming Score atau yang disebut label based accuracy sebagai evaluasi model. Yang memiliki rumus
Untuk lebih jelasnya dapat melihat jurnal ElKafrawy et al., (2015) pada link berikut.
